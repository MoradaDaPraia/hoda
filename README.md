# Hoda: Um Organizador de Tarefas por Projeto

## Setup do Projeto

Em Linux:

```sh
python -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
```

Em Windows (no cmd):

```sh
python -m venv .venv
.venv\Scripts\activate.bat
python -m pip install -r requirements.txt
```
